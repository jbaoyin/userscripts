# 🔍 聚合搜索

整合 DuckDuckGo、Bing（cn.bing.com）、Google 三个搜索引擎的油猴脚本，提供统一的悬浮工具栏，支持一键切换引擎、自动翻页、暗色模式、回到顶部。

## 功能

- **一键切换引擎**：点击悬浮栏上的引擎名称，用当前关键词跳转到另一个引擎重新搜索
- **自动翻页**（仅 Bing / Google）：滚动到页面底部自动加载下一页并拼接显示
- **一键回到顶部**：右下角 ⬆ 按钮快速回到页面顶部
- **暗色模式**：跟随系统浅色/深色外观自动切换
- **悬浮栏可拖动**：按住顶部标题栏拖到顺手的位置

## 安装

先装 [脚本猫 ScriptCat](https://scriptcat.org/) 浏览器插件，然后点击安装：

👉 [点击安装](https://github.com/jbaoyin/aggregate-search/raw/refs/heads/main/aggregate-search.user.js)

## 使用说明

1. 安装后在 DuckDuckGo / cn.bing.com / Google 搜索结果页会出现左侧悬浮栏
2. 点击栏上的引擎名称即可用同一关键词切换到另一个引擎
3. Bing / Google 悬浮栏中有"翻页：ON/OFF"开关，可手动关闭自动翻页
4. DuckDuckGo 未提供自动翻页
